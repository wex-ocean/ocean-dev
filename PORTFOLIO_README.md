# Ganesh Bahadur Thapa - Immersive Portfolio

A high-performance, futuristic React-based portfolio website showcasing WordPress development expertise through immersive 3D interactions, advanced motion design, and premium UI/UX.

## üöÄ Features

### Visual Design
- **Futuristic Dark Theme**: Pure black background with deep blue to vibrant blue gradients
- **Glassmorphism Effects**: Frosted glass cards with backdrop blur and soft borders
- **3D WebGL Background**: Animated particle system in the hero section
- **Gradient Text Effects**: Eye-catching gradient text for headings
- **Smooth Animations**: GSAP-powered scroll-triggered animations throughout

### Interactive Elements
- **Custom Cursor**: Magnetic cursor with scale effects on hover (desktop only)
- **Animated Header**: Transforms from full-width to floating pill on scroll
- **3D Hover Cards**: Project cards with tilt effects and depth shadows
- **Timeline Animation**: Scroll-linked experience timeline
- **Smooth Scrolling**: Physics-based smooth scrolling with Lenis

### Sections
1. **Hero**: Cinematic entrance with 3D particle background
2. **About**: Personal introduction with stats and glassmorphic design
3. **Skills**: 9 skill cards with staggered animations
4. **Experience**: Vertical timeline with 3 work experiences
5. **Projects**: 6 featured projects with live links
6. **Contact**: Interactive form with validation
